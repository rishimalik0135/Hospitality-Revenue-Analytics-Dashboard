# Hospitality Revenue Analytics Dashboard — Power BI

## Project Overview
Multi-view Power BI dashboard analysing hotel revenue performance across 
8 properties in 3 cities (Bangalore, Mumbai, Hyderabad) using 3 months 
of booking data.

## Tech Stack
- Excel - Data Source
- Power BI Desktop — data modelling and dashboard
- Power Query — data transformation and cleaning
- DAX — calculated measures and KPIs
- Power BI Service — publishing and sharing

## Key Metrics Tracked
- RevPAR (Revenue Per Available Room)
- ADR (Average Daily Rate)
- Occupancy %
- Realisation %
- DSRN, DBRN, DURN
- Cancellation %
- Average Guest Rating

## Key Findings
- Luxury category generates 61.62% of total revenue vs 38.38% Business
- Weekend RevPAR (7,972) outperforms weekday RevPAR (7,083) across all properties
- Total portfolio revenue: 1.69bn | Overall occupancy: 57.8% | Avg ADR: 12,696
- Direct offline booking channel shows highest realisation % at 70.6%
